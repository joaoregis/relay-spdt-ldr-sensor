# Example Relay SPDT + LDR Sensor
Example with Relay SPDT and LDR Sensor to turn on an light when room stay dark

# Dependencies:
- Arduino UNO
- C/C++ Language
- Relay SPDT
- LDR Sensor
- Protoboard
- 1 Resistor - 300 Ohms
- 1 Resistor - 1 KOhm
- Wires
